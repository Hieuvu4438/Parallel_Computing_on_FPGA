\begin{abstract}
This study presents a hierarchical embedded system that uses respiratory sound features to diagnose COPD and other respiratory disorders with accuracy and energy efficiency suitable for mobile and embedded devices. The system segments lung sounds into breathing cycles and extracts time- and frequency-domain features across four layers. It combines Random Forest classifiers with a neural network trained for disease classification. The first layer processes recording-level information; the second and third layers classify fixed-length segment features using Random Forest models; and the fourth layer converts breathing patterns into hybrid spectrogram images processed by a knowledge-distilled convolutional neural network (CNN). This design enables early classification without executing all four stages for every sample, thereby using FPGA parallelism more efficiently. Implemented on the Xilinx Ultra96-V2 FPGA development board, the system achieves 96.56\% average accuracy across three diagnostic categories (Healthy, COPD, and Non-COPD) and reduces latency and model size through QAT. These results demonstrate the potential of the proposed method for respiratory disorder diagnosis in real-world clinical applications.
\end{abstract}
