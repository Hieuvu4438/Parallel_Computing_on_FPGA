\begin{abstract}
This research focuses on designing a hierarchical embedded system that leverages respiratory sound features for diagnosing COPD and other respiratory disorders, aiming for high accuracy and energy efficiency suitable for mobile or embedded devices. The system segments lung sounds into individual breathing cycles and extracts key respiratory features across time and frequency domains, structured into four layers. It utilizes a combination of the Random Forest algorithm and an Artificial Neural Network, both of which are fine-tuned with the dataset for disease classification. The first layer handles record information, the second and third layers manage features extracted from fixed-length sound segments, classified using Random Forest, while the fourth layer converts breathing patterns into Spectrogram images using Wavelet Transform, which are then processed by the Artificial Neural Network for diagnosis. This design allows the system to classify without completing all four stages simultaneously, effectively utilizing the parallel computing power of the FPGA. Implemented on the Xilinx PYNQ-Ultra96-V2 FPGA development board, the system not only achieves a 98.81\% accuracy rate across five disease classes, Healthy, Pneumonia, URTI, Bronchiectasis, and COPD, but also reduces energy consumption by 52.5\% compared to traditional CPU-GPU methods. This study confirms the effectiveness of the proposed method in diagnosing respiratory disorders, highlighting its potential for real-world clinical applications.
\end{abstract}
